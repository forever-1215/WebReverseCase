# Web逆向案例分享

本项目收集整理了一些常见的Web逆向案例，主要涉及JS逆向、滑块验证破解等技术，仅供学习交流使用。

## 📚 项目简介

本仓库专注于分享Web逆向工程的实战案例，通过分析真实网站的加密算法、滑块验证等技术，帮助开发者更好地理解前端安全机制。所有案例均已脱敏处理，仅用于技术研究和学习目的。

## 🛠️ 技术栈

- **Python 3.x**
  - requests - HTTP请求
  - Pillow/OpenCV - 图像处理
  - selenium - 浏览器自动化（如需要）

- **JavaScript**
  - 加密算法分析（AES、RSA、MD5等）
  - 混淆代码还原
